说明下BrainFuck语言如何欺骗windows输出中文

我们来输出“飞飞”两个字。

我们知道简体中文操作系统使用的编码是GB2312，当然也可以自己写个小程序输入飞飞两个字看它在内存中的值。

然后将这些值输出就好，“飞”字的编码为：B7C9，然后我们来看brainfuck部分的代码：

```
<[-]>[-]<+++++ +++++ +   [> +++++ +++++ + +++++<-]>++ +++++.
<[-]>[-]<+++++ +++++ +++ [> +++++ +++++ + +++++<-]>-- -----.
 
<[-]>[-]<+++++ +++++ +   [> +++++ +++++ + +++++<-]>++ +++++.
<[-]>[-]<+++++ +++++ +++ [> +++++ +++++ + +++++<-]>-- -----.
 
<[-]+++++ +++++.
 
<[-]>[-]<++[> +++++ +++++ +++ +++<-]>.
```

稍微说明下，每一行是操作一个字符的值，倒数第二行为输出换行，最后为输出空格！
